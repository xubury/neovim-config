# My neovim config

My personal neovim config out of the box.

## Requirements

Install the following program to configure properly:

### Basic

-   Any terminal that support guicolor
-   git
-   cmake
-   wget
-   fzf sed ripgrep
-   node npm yarn
-   cargo
-   go
-   nerdfonts

### GUI related

-   neovide
-   Sarasa Gothic NF (Optional, change setting.lua's guifont accordingly)

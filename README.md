# Neovim config

My personal neovim config out of the box.

## Requirements

Install the following program to configure properly:

### Basic

-   Any terminal that support guicolor
-   git
-   cmake
-   wget
-   fzf sed ripgrep
-   node npm yarn
-   cargo
-   go
-   nerdfonts

### GUI related (Optional)

-   neovide
-   Sarasa Gothic NF (Change vim.g.guifont accordingly)
